#ZfCors

Provides CORS headers for your application responses.

### Installation
----------------

Add `ZfCors` to your `composer.json`:

~~~json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/Punchkick-Interactive/ZfCors.git"
        }
    ],
    "require": {
        "punchkick-interactive/ZfCors": "dev-master"
    }
}
~~~
